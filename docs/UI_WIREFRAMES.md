## Accessibility Requirements

1. **WCAG 2.1 AA Compliance**
2. **Keyboard Navigation**: All interactive elements focusable
3. **Screen Reader Support**: ARIA labels, roles, live regions
4. **Color Contrast**: Minimum 4.5:1 for text, 3:1 for large text
5. **Focus Indicators**: Visible focus rings on all interactive elements
6. **Skip Links**: Skip to main content on all pages
7. **Form Labels**: All inputs have associated labels
8. **Error Messages**: Clear, actionable error states
